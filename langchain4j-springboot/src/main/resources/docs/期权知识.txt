期权学习笔记

Variation margin 盈亏保证金，变动保证金

call option，看涨期权 - 认购（在未来的某一个时间，以约定的价格买入-锁定买入价）涨得越多，赚得越多
pull option，看跌期权 - 认沽（在未来的某一个时间，以约定的价格卖出-锁定卖出价）跌得越多，赚得越多
exercise price 执行价格 或 strike price 敲定价格+
expiration date 到期日 或 maturity 期限

基金类型：
股票型基金 混合型基金 债券型基金 货市型基金

按标的资产划分：
股票期权、股指期权、外汇期权、利率期权、商品期权等

按到期日划分：
American Option：在到期日之前任何时间都可以选择行使期权
European Option：只能在到期日才能选择是否行使期权
PS：欧式期权和美式期权相比，欧式期权本少利大，但在获利的时间上不具灵沽性，美式期权虽然灵活，但付费十分昂贵,因此，国际上大部分的期权交易都是欧式期权。

按行权价格划分：
    平值期权（At-The-Money）：标的物价格与行权价相同。
    实值期权（In-The-Money）：标的物价格与行权价存在行权利润。
        Call：行权价<当时标的物的价格
        Put：行权价>当时标的物的价格
    虚值期权（Out-of-The-Money）：标的物价格与行权价没有行权利
        Call：行权价>当时标的物的价格
        Put：行权价<当时标的物的价格

期权价 = 内涵价值 + 时间价值
    内涵价值（IntrinsicValue）
    实值期权：标的资产价格与行权价格的差虚值期权：内涵价值为
    时间价值（TimeValue）
    期权价减内涵价值的剩余部分

Black - Scholes模型是目前国际上大多数机构使用的期权定价方法

影响期权的价格因素：证券的价格、时间、波动率以外，还有行权价格、无风险利率，股息率。
    标的价格：标的价格上涨，认购期权（Cal1）的价值上升，认沽期权（Put）的价值下跌。
    时间：距离到期时间越短，期权价值越低。
    波动率：波动率越小，期权价值越低。（期权的市盈率）
    行权价格：
        1. 对于认购期权，行权价格越高，转向实值期权的可能性越小，权利金就相应降低。
            注： 假如标的物当前价格是1元，未来以168元价格买入的可能性非常小，权利金要求就低
        2。对于认沽期权，行权价格越高，转向实值期权的可能性越大，权利金就相应提高。
            注： 假如标的物当前价格是18元，未来以188元价格卖出的可能性非常高，权利金要求就高
    无风险利率（通常为国债利率）是影响期权价值最复杂的一个因素：当无风险利率上升时，未来行权价的现值会降低，意味着
        1. 认购期权的买方需要准备的行权资金的现值变少了，这份权力会更值钱，认购期权（Cal）的价格会上升; 
        2．认沽期权的买方，目的是以行权价的资金出售标的的股票，无风险利率变，意味着失去的机会成本变大，这份权力就会变得不值钱，认沽期权（Put）的价格会下。
        3. 如果无风险利率上升，标的资产价格的预期增长率可能上升，而期权买方未来可能收到的现金流的现值将下降，这两个因素都使认沽期权（Put）的价值下降。因此，其他因素不变，无风险利率上升，认购期权价值上升，认沽期权价值下
    股息率：股息率越高，认购期权的价值会变小，认沽期权的价值会变大。
        一般来说，如果在期权到期日前，对标的股票进行了分红，那么除息日后，标的股票价格会下跌，标的价格下跌，认购期权的价值下降，认沽期权的价值上升。

买入认购开仓（新建头寸）－卖出认购平仓（了结头寸）
买入认沽开仓（新建头寸）－卖出认沽平仓（了结头寸）

看不涨卖认购    -   看不跌卖认估    -   看大跌买认沽    -   看大涨买认购

买入开仓 - 付出权利金 - 最多亏掉权利金（不行权）
卖出平仓 - 收权利金 - 利润有限

当标的出现上涨时，买入认购期的投资者获得收益。
    标的价格涨的越多，买入认购期权的收益也就越大。买入的投资者承担的损失是有限的，如果标的价格下跌，低于了行权价格，选择不行权，最大损失就是全部权利金。
当标的出现上涨时，卖出认购期权的投资者出现亏损。
    标的价格涨的越多，卖出认购期权的亏损就越大。卖出认购的投资者获得的收益是有限的。
当标的出现下跌时，买入认沽期权的投资者获得收益。
    标的价格跌得越多，买入认沽期权的收益也就越大。当标的证券的价格跌到元时收益达到最大值，买入认沽的投资者承担的损失是有限的。标的价格上涨，选择不行权，最大损失即为全部权利金。
当标的出现下跌时，卖出认沽期权的投资者出现亏损。标的价格跌的越多，认沽期权的卖方亏损就越大。
    卖出认沽的投资者获得的收益是有限的，最多为全部权利金，但下行风险无限。

综合以上内容，我们再来记忆期权手指操和期权基本交易的关系：
左手拇指向上：（）买入Cal1（看涨期权、买权），看涨，支付权利金，风险有限，获利无限。
右手拇指向上：（）买入Put（看跌期权，卖权），看跌，支付权利金，风险有限，获利无限。
左手拇指向下：（）卖出Cal1（看涨期权、买权），看不涨，收取权利金，获利有限，风险无限
右手拇指向下：（）卖出Put（看跌期权、卖权），看不跌，收取权利金，获利有限，风检无限。

买方策略：
    投保人、缴纳权利金，只有权力没有义务，不需要缴纳保证金。
    风险：价值归零风险、高溢价风险、流动性风险、行权交割风险。
卖方策略：
    保险公司、承担义务，收取权利金，需要缴纳保证金。
    风险：保证金风险、巨额亏损风险、流动性风险、行权交割风险。


保险策略
    保险策略的最大损失-标的证券的买入价格+认沽期权权利金-认沽期权的行权价，
    盈亏平衡点-标的证券买入价格+认沽期权权利金。
    最大收益上不封顶

领口策略 -- 在持有保险策略的同时，卖出认购期权，以认购期权所得的权利金收入降低整个保险策略的成本。
    领口策略的构建成本低于保险策略，但组合收益有上限。
        最大损失-（标的证券购买价格-认沽期权行权价）+（认沽期权权利金-认购期权权利金），
    盈亏平衡点-购买标的证券价格+认沽期权权利金-认购期权权利金，
    最大收益－（认购期权行权价-标的证券购买价格）+（认购期权权利金-认沽期权权利金）。

备兑开仓的盈亏分析
    备兑开仓策略的最大损失-标的证券买入成本-权利金（当标的证券价格为时），盈亏平衡点-标的证券买入成本-权利金，最大收益权利金+认购期权行权价-标的证券买入成本。
    当期权合约到期时，若标的证券价格低于盈亏平衡点，标的证券价格下降越多，亏损越大。
    若标的证券价格处于盈亏平衡点与行权价之间，标的证券价格越接近行权价，则组合收益越大。
    若标的证券价格大于行权价，达到备兑开仓策略最大收益，此时卖出的认购期权处于实值状态，期权买方提出行权后按照行权价购买标的证券。
    整个备兑开仓组合的收益率，等于（权利金收入+股票价差收入）／净成本（非年化）。
    
合成股票策略（买入认购+卖出认沽）
    合成股票多头策略的构建方法为买入一份认购期权，同时卖出一份具有相同行权价相同到期日的认沽期权，当投资者强烈看好标的证券未来走势时，可通过这种交易策略来实现低成本做多后市。

跨式和勒式策略（买认购+买认沽OR卖认购+卖认沽）
    “跨式组合”付出了双重的权利金，成本一般较高
    可以利用认购、认沽期权构造出当标的价格出现大幅波动时收益无限、损失有限的买人跨式和勒式策略。
        策略的构成很容易记忆一“三相同”，即数量相同、行权价相同、到期日相同。
        买入跨式策略的最大损失一买入认购期权权利金+买入认沽期权权利金，理论上看最大收益无上限，只有当标的证券价格波动超过盈亏平衡点，该策略才会有盈利。
    相反，当预期后市为小幅盘整行情时，则可通过卖出跨式和勒式策略来获得收益。
        卖出相同数量，相同行权价的同月认购和认沽期权来构成卖出跨式策略，
        卖出跨式策略的最大收益-卖出认购期权权利金+卖出认沽期权权利金，最大损失无限

牛市，熊市价差策略（买入低行权价认购期权 + 卖出高行权价认购期权－牛市差价 OR 买高行权价认沽期权+卖出低行权价认洁期权 = 熊市价差）
    当投资者对未来行情适度看涨时，可以运用牛市价差策略。策略构建口为“买低卖高“即买入低行权价的期权，再卖出一个高行权价的同月同类型期权。
    买入牛市认购价差策略具有收益有限、损失有限的特点.
        策略组合的最大损失为构建成本-较低行权价（K1）的认购期权权利金-较高行权价（K2）的认购期权权利金，
        最大收益-较高行权价（K2）-较低行权价（K1）－构建成本。

鹰式和蝶式策略因为要牺性一部分利润来换取有限的亏损，在实际交易过程当中有些行情结构并不会经常用到。

我们看到在英文标注中有 Buy、Sell、Long、Shot 的区别，例如：Buy Strangle和Long Strangle的中文都译为买入宽跨式期权策略。
那么Buy、Sell、Long、Shot有什么不同呢？Buy和Sell表达的是交易的行为，买入或者卖出，Long（看多）和Shot（看空）表达的是对市场的看法。
例如：Buy一个Put行为上是买入，看法上是Shot市场。

期权交易参数
    Delta：衡量标的的资产价格变化对期权价格的影响。
        Delta = 期权价格变化/标的资产价格变化+
        也即
        新的期权价格=原期权价格+Delta*标的资产价格变化

    Gamma：衡量标的的资产价格变化对Delta的影响，间接度量了标的资产价格变化对期权价格的二阶影响。
        Gamma = 原Delta + Gamma*标的资产价格变化+
        新的期权价格 = 原期权价格 + Delta*标的资产价格变化 +（1/2*Gamma*标的资产价格变化²）

    Vega：表示隐合波动率，衡量隐含波动率变化对期权价格的影响，直接影响期权价。
        Vega = 期权价格变化/波动率的变化
        新的期权价格 = 原期权价格 +Vega*波动率变化

    Theta：衡量到期时间变化对期权价格的影响，不论时看涨期权还是看跌期权，到期时间越长则期权的价值越高，随着时间的流逝，期权的价值则不断下降。
        Theta = 期权价格变化/到期时间变化（passage of time）
        新的期权价格 = 原期权价格 + Theta*到期时间变化

    Rho：衡量利率变化对期权价格的影响。在其它因素不变的前提下，距高到期日的时间越长，外汇期权的Rho就越大。
        Rho对期权价格的影响，也就是期权到期支付或者收取现值的变化是一样的。
            如果距离期权合约到期日的时间不变，无风险利率上升，认购期权到期支付的现值就会下降，期权的价格就会上升，所以，对于认购(call)期权，Rho大于零。
            而对于认沽(pull)期权，无风险利率上升会导致到期收取的现值下降，期权的价格同样也会下降，因此，对于认沽期权而言，Rho是小于零。
        Rho = 期权价格的变化／无风险利率的变化
        新的期权价格 = 原期权价格+Rho*利率变化
